# Contributing

DeplyzeGPT is open source for transparency and learning purposes. The codebase is shared so others can study the architecture, learn from the implementation, and understand how the product works.

## Pull Requests

We are **not currently accepting external pull requests**. The project is actively developed by the core team, and the public repository serves as a reference rather than a collaborative development space.

## Issues and Questions

If you find a bug, have a question, or want to suggest an improvement, please open a [GitHub Issue](https://github.com/dyglo/vision-language/issues). We read all issues and will respond when possible.
